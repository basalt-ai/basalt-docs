# Making your first API request

You are now ready to make your first call with the Basalt API. Below, you will discover how to get your first prompt. {{ className: 'lead' }}

## Prerequisites

- You have an API key. If you don't have it, you can create one in the your Settings.

## Get your prompt

<CodeGroup>

```ts
// Import the Basalt SDK
import Basalt from '@basalt-ai/sdk'

// Initialize the SDK with your API key
const basalt = new Basalt(token)

// Get your prompt
await basalt.prompt.get('my-prompt-slug')
```

```python
# Import the Basalt SDK
from basalt import Basalt

# Initialize the SDK with your API key
basalt = Basalt(token)

# Get your prompt
basalt.prompt.get('my-prompt-slug')
```

```bash {{ title: 'cURL' }}
curl -G https://api.getbasalt.ai/prompts/my-prompt-slug \
  -H "Authorization: Bearer {token}" \
```
