# Basalt

Repo for the [Basalt documentation](https://docs.getbasalt.ai) based on [Tailwind Protocol](https://tailwindui.com/templates/protocol)

## Getting started

To get started with this repo, first install the npm dependencies:

```bash
npm install
```

Next, run the development server:

```bash
npm run dev
```

Finally, open [http://localhost:3000](http://localhost:3000) in your browser to view the website.

## Customizing

You can bring some modifications by editing the files in the `/src` folder. The site will auto-update as you edit these files.
